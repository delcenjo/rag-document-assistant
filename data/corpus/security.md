# Nimbus Notes — Security

Security is handled in line with industry best practice and audited by external
parties.

## Encryption

- Data is encrypted in transit with TLS 1.3.
- Data is encrypted at rest with AES-256.
- Optional end-to-end encryption is available on Pro and higher plans.

## Compliance and access control

- SOC 2 Type II certified and GDPR compliant.
- Two-factor authentication via TOTP authenticator apps.
- Data is stored in data centres located in the EU and the US, selectable per
  workspace.

## Reliability

- Encrypted backups run daily and are retained for 30 days.
- The platform is penetration-tested annually by an independent firm.
