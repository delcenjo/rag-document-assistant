# Nimbus Notes — Product Overview

Nimbus Notes is a cloud-based note-taking application for individuals and teams.
It combines rich-text and Markdown editing with real-time collaboration, so
several people can edit the same note at once.

## Key features

- Rich-text and Markdown editing with live preview.
- Real-time collaboration and inline comments.
- Offline mode on desktop and mobile, with automatic sync once back online.
- Full-text search across all notebooks.
- Organisation through notebooks, tags and nested pages.
- Cross-platform clients for web, Windows, macOS, Linux, iOS and Android.

## Availability

Nimbus Notes launched in 2019 and is available in 12 interface languages. The
service is offered as a hosted SaaS product; an on-premise deployment is
available on the Enterprise plan.
